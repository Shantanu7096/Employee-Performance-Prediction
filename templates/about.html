Project Overview

Objectives

This project aims to leverage Machine Learning (Random Forest Regressor) to predict the productivity of workers in the garment industry. By analyzing historical data, we provide insights into how variables like team size, targeted goals, and overtime affect actual output.

Key Insights

  • Data-Driven: Analysis of 1,200+ production records.
  • Accuracy: Uses SMV (Standard Minute Value) to calculate precise effort.
  • Optimization: Identifies "Idle Men" and "Idle Time" bottlenecks.

Developed for the Internship Performance Prediction Module.
